# D2-grafo-fases — Pipeline = grafo dirigido con fases

- **Estado:** aceptada (confirmar/impugnar en F0)
- **Fecha:** 2026-07-16
- **Origen:** plan maestro §1

## Contexto
Un swarm conversacional entre agentes es no determinista y caro de depurar.

## Decisión
El pipeline es un grafo de fases fijas; la creatividad vive dentro de cada agente, no en la coordinación.

## Consecuencias
Determinismo, coste predecible, debugging. Menos emergencia creativa entre agentes (aceptable).
